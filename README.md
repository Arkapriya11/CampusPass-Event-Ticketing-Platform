# CampusPass – Interactive Event Registration & Ticketing Platform

## Introduction

CampusPass is a full-stack event registration, ticketing, and attendee management platform designed to streamline the complete event lifecycle for colleges, universities, conferences, workshops, and large-scale institutional events.

The platform combines secure ticketing, online payment processing, QR-based attendee verification, automated email communication, role-based access control, and real-time analytics into a unified system. In addition to traditional event management capabilities, CampusPass features an immersive and visually engaging 3D landing experience designed to enhance user interaction and modernize the registration workflow.

Built using React, Node.js, Express, Supabase, and Razorpay, the platform demonstrates the integration of modern frontend experiences with scalable backend services and secure transaction processing.

## Core Features

### Interactive 3D User Experience

CampusPass incorporates an immersive landing page featuring interactive 3D elements and modern visual effects to create an engaging user experience.

Features include:

* Interactive 3D scene rendering
* Animated visual components
* Dynamic particle effects
* Modern glassmorphism-inspired design
* Responsive user interface across devices
* Enhanced user engagement through visual storytelling

This design approach improves the overall user experience while showcasing modern frontend engineering practices.

---

### Event Lifecycle Management

The platform enables administrators to create, manage, and monitor events through a centralized dashboard.

Capabilities include:

* Event creation and modification
* Ticket category management
* Pricing configuration
* Event scheduling
* Event status tracking

---

### Role-Based Access Control (RBAC)

CampusPass implements a secure role-based architecture supporting three operational roles:

#### Administrator

* Manage events
* Manage users
* Access analytics and reports
* Monitor platform operations

#### Ticket Seller

* Register participants
* Process ticket sales
* Manage attendee information
* Generate event tickets

#### Ticket Checker

* Validate attendee tickets
* Perform event check-ins
* Monitor entry operations
* Prevent duplicate admissions

### Digital Ticketing System

Features include:

* Auto-generated unique ticket serial numbers
